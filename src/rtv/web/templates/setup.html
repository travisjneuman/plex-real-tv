SSH (Remote Server)

{{ "Enabled" if config.ssh.enabled else "Disabled" }}

Only needed if this app runs on a different machine than your Plex server. For example, if you're running plex-real-tv on your laptop but Plex Media Server lives on a dedicated server in your closet, SSH lets this app manage commercial files on the remote server (upload, scan directories, etc.) without needing to copy files manually. If Plex is on this same machine, you can leave SSH disabled.

IP address or hostname of your Plex server machine.
SSH port (almost always 22, the default).
The user account on the Plex server to log in as (e.g. "Administrator", "admin").
Path to your SSH private key file on this machine. Leave empty to use the default key or SSH agent.
The folder on the remote Plex server where commercial video files are stored. This is the path as it appears on the server itself (e.g. "F:\Commercials" on Windows, "/media/commercials" on Linux).
